### Folder Structure Information

##### Designs
Fully skinned wireframes that will be used directly as assets for the front end development.

##### Wireframes
Basic design outline of the application that incorporates the requirements in a visual form. Not all requirements can be visually described in this document especially in the case of serverside or database features

##### Mockups
Basic design that incorporates all the information of the wireframe but also includes some design work. Often times this should be a clickable hybrid between a wireframe and design
